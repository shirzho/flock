Flock

This is our current deployment for Flock our crime heat map application.

We will be using Zeit.co's Now to deploy all versions of our application as we progress

Flock lives in a Node JS application running with Express - a web application framework


Click here to see our map demo using the leaflet JS API

Currently, we are able to have markers and shapes on the map as well as being able to create a heat map using an array input of coordinates


add "/_src" to the end of the current site URL to see our file directory and everything we are currently running on and built upon
